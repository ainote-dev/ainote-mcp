## 개요

AI Note MCP(Model Context Protocol)는 Claude Desktop에서 AI Note의 할 일을 관리할 수 있게 해주는 통합 인터페이스입니다. 자연스러운 대화를 통해 할 일을 조회, 생성, 수정, 삭제할 수 있습니다.

## 기본 기능

### 📋 할 일 조회 (list_tasks)

할 일 목록을 다양한 조건으로 조회할 수 있습니다.

**기능:**
- 전체 할 일 목록 조회
- 상태별 필터링 (진행중/완료)
- 키워드 검색
- 최대 500개까지 조회 가능

**명령 예시:**
- "내 할 일 목록을 보여줘"
- "오늘 해야 할 일이 뭐야?"
- "완료된 할 일을 보여줘"
- "아직 안 끝난 일들만 보여줘"
- "중요한 할 일만 보여줘"
- "'보고서'가 포함된 할 일을 찾아줘"
- "'프로젝트'라는 단어가 들어간 할 일 있어?"

### ➕ 할 일 생성 (create_task)

새로운 할 일을 추가할 수 있습니다.

**기능:**
- 할 일 내용 입력
- 중요도 설정 (⭐ 표시)
- 마감일 설정
- 카테고리 지정

**명령 예시:**
- "새 할 일 추가: 프레젠테이션 준비하기"
- "할 일 추가해줘: 장보기"
- "중요한 할 일 추가: 세금 신고하기"
- "별표 표시해서 '병원 예약' 할 일 추가"
- "개인 카테고리에 '운동하기' 할 일 추가해줘"
- "업무 카테고리에 '회의 준비' 추가"
- "내일까지 해야하는 '보고서 제출' 할 일 추가"
- "금요일까지 '프로젝트 마감' 할 일 만들어줘"

### ✏️ 할 일 수정 (update_task)

기존 할 일의 내용이나 상태를 변경할 수 있습니다.

**기능:**
- 할 일 내용 수정
- 중요도 변경
- 완료/미완료 상태 변경

**명령 예시:**
- "할 일 '보고서 작성'을 '보고서 검토 및 제출'로 변경해줘"
- "'회의 준비'를 '팀 회의 자료 준비'로 수정"
- "'회의 준비' 할 일을 중요 표시해줘"
- "'운동하기'에 별표 추가해줘"
- "'프로젝트 완료' 할 일을 완료 처리해줘"
- "'보고서 제출' 끝났어"
- "'운동하기' 할 일을 미완료로 되돌려줘"
- "'청소하기' 아직 안 끝났어"

### 🗑️ 할 일 삭제 (delete_task)

불필요한 할 일을 삭제할 수 있습니다.

**기능:**
- 할 일 영구 삭제 (soft delete 방식)

**명령 예시:**
- "'임시 메모' 할 일을 삭제해줘"
- "'테스트' 할 일 지워줘"
- "완료된 '구 프로젝트' 관련 할 일들을 삭제해줘"
- "'취소된 미팅' 할 일 제거해줘"

### 📁 카테고리 조회 (list_categories)

카테고리 목록과 각 카테고리의 할 일 개수를 확인할 수 있습니다.

**기능:**
- 전체 카테고리 목록 조회
- 카테고리별 할 일 개수 확인
- 카테고리 색상 정보

**명령 예시:**
- "내 카테고리 목록을 보여줘"
- "어떤 카테고리들이 있는지 알려줘"
- "카테고리별로 할 일이 몇 개씩 있어?"

## 고급 활용법

### 🌅 일일 계획 수립

아침에 오늘의 할 일을 확인하고 계획을 세울 때:

```
사용자: "오늘 뭐해야 하지?"
Claude: [오늘의 할 일 목록 표시]

사용자: "이 중에서 중요한 것부터 처리하고 싶어"
Claude: [중요도 순으로 정렬된 할 일 표시]

사용자: "첫 번째랑 세 번째 할 일을 오전에 끝내야겠다"
Claude: [해당 할 일들에 대한 상세 정보 제공]
```

### 📊 주간 리뷰

주말에 한 주를 돌아볼 때:

```
사용자: "이번 주에 뭘 했는지 정리해줘"
Claude: [이번 주 완료된 할 일 요약]

사용자: "못 끝낸 중요한 일들이 있나?"
Claude: [미완료된 중요 할 일 목록]

사용자: "다음 주로 미뤄야겠네. 메모 좀 추가해줘"
Claude: [할 일에 메모 추가 제안]
```

### 🏷️ 카테고리별 관리

특정 영역의 할 일만 집중적으로 관리:

```
사용자: "업무 관련 할 일들만 보여줘"
Claude: [업무 카테고리 할 일 목록]

사용자: "이 중에서 이번 주 안에 끝내야 하는 것들이 뭐야?"
Claude: [마감일 기준 필터링된 목록]

사용자: "개인 카테고리는 어때? 밀린 게 있나?"
Claude: [개인 카테고리의 미완료 할 일 표시]
```

### 🎯 프로젝트 관리

특정 프로젝트 관련 할 일들을 추적:

```
사용자: "'신제품 출시' 관련된 할 일들 다 보여줘"
Claude: [키워드로 필터링된 할 일 목록]

사용자: "진행 상황이 어때?"
Claude: [완료/미완료 비율과 함께 상태 요약]

사용자: "마케팅 관련 할 일 하나 더 추가해야겠다"
Claude: "'신제품 출시 - 마케팅 전략 수립' 할 일을 추가할까요?
```

### 🎾 활동 패턴 분석 & 맞춤 제안

완료된 할 일에서 패턴을 파악하여 관련 정보 제공:

```
사용자: "완료한 일들 중에 테니스 관련된 게 있나?"
Claude: [테니스 관련 완료 할 일 표시]
        최근 매주 토요일 오전 10시에 테니스를 치신 기록이 있네요.

사용자: "이번 주말 날씨 어때?"
Claude: 최근 완료하신 할 일들을 보니 테니스를 정기적으로 치고 계시네요! 
        특히 매주 토요일 오전 10시에 테니스 일정이 많이 보입니다.

        **이번 주말 날씨 예보** (테니스 치시는 시간대 중심으로):

        🎾 **토요일 (2월 3일)**
        - 오전 10시: ☀️ 맑음, 18°C
        - 체감온도: 17°C, 습도: 55%
        - 바람: 북서풍 5km/h (미풍)
        - **테니스 추천도: ⭐⭐⭐⭐⭐ 완벽한 날씨!**

        🎾 **일요일 (2월 4일)**
        - 오전 10시: ⛅ 구름 조금, 16°C
        - 체감온도: 15°C, 습도: 65%
        - 바람: 남서풍 8km/h
        - 오후 3시부터 비 예보 (강수확률 70%)
        - **테니스 추천도: ⭐⭐⭐⭐ 오전만 추천**

        💡 **테니스 일정 제안:**
        토요일이 날씨가 더 좋으니 토요일 오전에 테니스를 즐기시는 것을 추천드립니다.
        일요일은 오후에 비가 올 예정이니, 오전 일찍 마치시는 것이 좋겠습니다.

사용자: "그럼 이번 주도 테니스 일정 추가해줘"
Claude: "토요일 오전 10시 테니스" 할 일을 추가했습니다.
        이번 주말도 좋은 테니스 되세요! 🎾
```

#### 🏃‍♀️ 운동 루틴 관리 예시

```
사용자: "운동 관련 완료한 일들 보여줘"
Claude: [헬스장, 러닝, 요가 등 운동 관련 완료 할 일 표시]
        매주 화, 목, 토요일에 헬스장을 다니시는 패턴이 보이네요.

사용자: "이번 주 운동 계획 어떻게 하지?"
Claude: 기존 패턴을 보면 화목토 헬스장 루틴을 유지하고 계시네요.
        이번 주 화요일과 목요일 헬스장 일정 추가할까요?
        토요일은 날씨가 좋으니 야외 러닝도 좋을 것 같습니다.

사용자: "좋아, 화목은 헬스장, 토요일은 한강 러닝으로 추가해줘"
Claude: 3개 운동 일정 추가했습니다:
        - 화요일 오후 7시: 헬스장 운동
        - 목요일 오후 7시: 헬스장 운동  
        - 토요일 오전 9시: 한강 러닝
```

#### 📚 학습 패턴 분석 예시

```
사용자: "공부 관련 완료한 일들 정리해줘"
Claude: [어학, 자격증, 독서 등 학습 관련 완료 할 일 표시]
        영어 공부를 꾸준히 하고 계시네요. 특히 매일 오전 30분씩 영단어 학습을 하신 기록이 있습니다.

사용자: "이번 주는 어떻게 계획하지?"
Claude: 기존 패턴을 유지하면서 조금 더 체계적으로 해보시는 건 어떨까요?
        
        **제안 학습 계획:**
        - 월~금: 오전 30분 영단어 (기존 유지)
        - 화, 목: 저녁 1시간 영문법
        - 주말: 영어 드라마 시청 (재미있게 공부)

사용자: "좋아, 그렇게 일정 추가해줘"
Claude: 이번 주 영어 학습 일정 추가 완료!
        꾸준히 하고 계신 만큼 분명 실력이 늘고 있을 거예요. 화이팅! 📚
```

## 자연어 명령 예시

### 🗣️ 다양한 표현 방식

같은 작업도 여러 방식으로 요청할 수 있습니다:

**할 일 조회:**
- "할 일 뭐 있지?"
- "오늘 일정 알려줘"
- "내가 뭐 해야 되더라?"
- "할 일 목록 보자"

**할 일 완료:**
- "'운동' 끝"
- "'운동하기' 완료"
- "'운동' 했어"
- "'운동' 체크"

**중요 표시:**
- "이거 중요해"
- "별표 붙여줘"
- "중요 표시해"
- "⭐ 추가"

### 💬 문맥 기반 대화

이전 대화 내용을 기억하여 자연스러운 대화 가능:

```
사용자: "내 할 일 보여줘"
Claude: [할 일 목록 표시]

사용자: "첫 번째 거 끝났어"
Claude: "'보고서 작성' 을 완료 처리했습니다.

사용자: "두 번째는 뭐였지?"
Claude: 두 번째 할 일은 '회의 준비'입니다.

사용자: "그거 내일로 미뤄야겠다. 메모 좀 수정해줘"
Claude: 어떤 내용으로 수정할까요?
```

## 제한사항

### ❌ 현재 지원하지 않는 기능

1. **카테고리 관리**
   - 카테고리 생성/수정/삭제 (앱에서만 가능)
   - 카테고리 순서 변경
   - 카테고리 색상 변경

2. **고급 할 일 기능**
   - 우선순위 설정 (중요도 ⭐만 지원)
   - 반복 할 일 설정
   - 알림/리마인더 설정
   - 할 일 공유/협업
   - 파일 첨부
   - 서브태스크(하위 할 일)
   - 태그 기능

3. **데이터 관리**
   - 할 일 백업/복원
   - 데이터 내보내기
   - 일괄 작업 (대량 삭제 등)

### 📊 사용 제한

- **일일 요청 제한**: 100회 (기본값)
- **일일 토큰 제한**: 100,000 토큰
- **최대 MCP 키 개수**: 사용자당 5개
- **한 번에 조회 가능한 할 일**: 최대 500개
- **할 일 내용 길이**: 최대 1000자
- **카테고리명 길이**: 최대 50자

## 활용 팁

### 💡 효율적인 사용법

1. **일괄 처리보다는 대화식 작업**
   ```
   ❌ "할 일 10개 한번에 추가해줘"
   ✅ "회의 관련 할 일 몇 개 추가하려고 해" → 대화하며 하나씩 추가
   ```

2. **구체적인 키워드 사용**
   ```
   ❌ "그거 완료해줘"
   ✅ "'보고서 작성' 완료해줘"
   ```

3. **카테고리 활용**
   ```
   "업무 카테고리에 추가: 프레젠테이션 준비"
   "개인 일정만 보여줘"
   ```

### 🎯 생산성 향상 팁

1. **일일 루틴 만들기**
   - 아침: "오늘 할 일 뭐야?"
   - 점심: "오전에 뭐 끝냈지?"
   - 저녁: "오늘 못 끝낸 중요한 일 있나?"

2. **주간 계획 수립**
   - 월요일: "이번 주 중요한 일정들 정리해줘"
   - 금요일: "이번 주 완료한 것들 요약해줘"

3. **프로젝트별 추적**
   - 일관된 키워드 사용 (예: [프로젝트A])
   - 정기적인 진행상황 확인

### 🔧 문제 해결

**할 일을 찾을 수 없을 때:**
- 더 구체적인 키워드 사용
- 다른 표현으로 검색
- 카테고리로 먼저 필터링

**명령이 인식되지 않을 때:**
- 더 명확한 동사 사용 (추가, 삭제, 수정, 완료)
- 할 일 제목을 정확히 명시

**응답이 느릴 때:**
- 검색 범위를 좁혀서 요청
- 불필요한 조건 제거
